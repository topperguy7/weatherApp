About WeatherApp


×

WeatherApp is a simple and responsive web application that provides real-time weather information for any city in the world.


It uses a free public Weather API to fetch live data such as temperature, weather condition, and atmospheric details.


The project is built using HTML, CSS, and JavaScript, focusing on API integration, DOM manipulation, and clean UI design.


This application demonstrates how client-side applications communicate with external APIs to deliver dynamic data.
